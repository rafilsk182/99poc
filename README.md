# 99Food — Design Tokens & Style Guide

Design tokens extraídos de produção de [99app.com/99food](https://99app.com/99food/).

## Arquivos

| Arquivo | Descrição |
|---|---|
| `99food-styleguide.html` | Style guide navegável — abrir direto no browser, sem dependências |
| `99food-tokens.json` | Tokens estruturados em JSON para uso em ferramentas de design/código |

## Como usar

```bash
# Abrir o style guide
open 99food-styleguide.html
```

## Tokens incluídos

- **Cores** — 90+ tokens semânticos (Orange, Yellow99, Neutral, Feedback, Transparências)
- **Tipografia** — Didi Sans, Didi Sans Flex BR, Montserrat · escala xs→7xl · 9 font-weights
- **Espaçamento** — base 4px, 24 steps
- **Border Radius** — xs (2px) → pill (9999px)
- **Sombras** — 8 níveis
- **Breakpoints** — sm:640 / md:768 / lg:1024 / xl:1280 / wide:1440 / 2xl:1536 / ultrawide:1940
- **Transições & Animações** — 12 keyframes, 3 easing curves
- **Z-Index** — escala completa

Tailwind CSS v4 + Next.js (App Router)
